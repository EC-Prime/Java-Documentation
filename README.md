# Java-Documentation
Java Technical Documentation Page A technical documentation page covering core Java concepts — built as part of the freeCodeCamp Responsive Web Design Certification.

## 🔗 Live Demo

java-documentations.netlify.app

---

## 📌 About

A responsive technical reference page documenting fundamental Java concepts including syntax, variables, data types, operators, control flow, and loops. Features a fixed sidebar navigation on desktop that collapses to a top navbar on mobile.

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, media queries, transitions)

---

## ✨ Features

- Responsive layout — sidebar nav on desktop, top nav on mobile
- Fixed navigation with smooth hover animations
- Styled code blocks with hover color transitions
- Clean section structure with anchor link navigation
- Custom footer branding

---

## 📚 What I Learned

- Building responsive layouts with Flexbox and `flex-direction` switching via media queries
- Fixing a sidebar with `position: fixed` and offsetting main content with `margin-left`
- Styling `<code>` blocks with `white-space: pre` for proper code formatting
- Using CSS `transition` for smooth hover interactions on nav links and code blocks
- Structuring a multi-section document with semantic HTML anchor navigation
